# Teclas de atalho VSCode


Nesse arquivo vamos começar a ver as teclas de atalho que uso no dia dia. Decidi anotar porque sempre esqueço depois de um tempo sem programar.

* Ctrl + Shift + P -> Paleta de comandos
* Ctrl + P -> Pesquisa de arquivos
* ***Usar o @ serve para pesquisar metodos e funções no arquivo***
* Para mover a seleção podemos usar o *alt* e seta para cima ou para baixo
* Ctrl + Shift + seta para cima ou para baixo -> Multicursor
* Ctrl + Shift + l -> Multicursor por seleção.
* Ctrl + Alt + seta para cima ou para baixo -> Clona seleção para cima ou para baixo (OBS: No linux mint, recomendo modificar as teclas de atalhos conflitantes)
